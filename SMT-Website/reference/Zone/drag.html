Name
Zone.drag()
Description

Translates the zone, its group, and its children

Syntax

drag()
drag(dragX, dragY)
drag(dragLeft, dragRight, dragUp, dragDown)
drag(dragLeft, dragRight, dragUp, dragDown, leftLimit, rightLimit, upLimit, downLimit)
drag(dragX, dragY, leftLimit, rightLimit, upLimit, downLimit)
drag(from, to)
drag(from, to, dragX, dragY)
drag(from, to, dragLeft, dragRight, dragUp, dragDown)
drag(fromX, fromY, toX, toY)
drag(fromX, fromY, toX, toY, dragX, dragY)

Parameters
dragX:
Whether to drag along the x-axis

dragY:
Whether to drag along the y-axis

dragLeft:
Allow dragging left

dragRight:
Allow dragging Right

dragUp:
Allow dragging Up

dragDown:
Allow dragging Down

leftLimit:
Limit on how far to be able to drag left

rightLimit:
Limit on how far to be able to drag right

upLimit:
Limit on how far to be able to drag up

downLimit:
Limit on how far to be able to drag down

from:
The Touch to drag from

to:
The Touch to drag to

fromX:
The x value to drag from

fromY:
The y value to drag from

toX:
The x value to drag to

toY:
The y value to drag to


Returns

void
