Name
KeyboardZone
Description

KeyboardZone is an implementation of an on-screen keyboard. Use its addKeyListener(Zone) method to allow other zones, such as TextZone, to listen/receive the keyboard input.

Syntax

KeyboardZone()
KeyboardZone(keysSentToApplet)
KeyboardZone(x, y)
KeyboardZone(x, y, keysSentToApplet)
KeyboardZone(name)
KeyboardZone(name, keysSentToApplet)
KeyboardZone(x, y, width, height)
KeyboardZone(x, y, width, height, keysSentToApplet)
KeyboardZone(name, x, y, width, height)
KeyboardZone(name, alpha)
KeyboardZone(name, keysSentToApplet, alpha)
KeyboardZone(x, y, width, height, alpha)
KeyboardZone(x, y, width, height, keysSentToApplet, alpha)
KeyboardZone(name, x, y, width, height, alpha)
KeyboardZone(name, x, y, width, height, keysSentToApplet, alpha)
KeyboardZone(name, x, y, width, height, keysSentToApplet, alpha, backgroundColor, keyColor, keyPressedColor, textColor)
KeyboardZone(name, x, y, width, height, keysSentToApplet, alpha, backgroundColor, keyColor, keyPressedColor, textColor, font)

Parameters
name:
- String: The name of the zone

x:
- int: X-coordinate of the upper left corner of the zone

y:
- int: Y-coordinate of the upper left corner of the zone

width:
- int: Width of the zone

height:
- int: Height of the zone

keysSentToApplet:
- boolean: Sets if the key input is sent to Applet

alpha:
- int: Transparency level for keyboard

backgroundColor:
- int: Background colour of the keyboard

keyColor:
- int: Colour of the keyboard keys

keyPressedColor:
- int: Colour of the keyboard keys when they are pressed down

textColor:
- int: Colour of the text in the keyboard keys

font:
- PFont: Font used to display text in the keyboard keys
