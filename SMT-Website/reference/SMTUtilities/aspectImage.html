Name
SMTUtilities.aspectImage()
Description

This method draws the image at (x,y) within the specified width/height parameters, but maintaining the aspect ratio of the image.

Syntax

aspectImage(g, image, x, y, width, height)
aspectImage(applet, image, x, y, width, height)

Parameters
g:
the graphics context to draw the image in

image:
the image to draw

x:
the x-coordinate of the position to draw it at

y:
the y-coordinate of the position to draw it at

width:
the width to draw the image within

height:
the height to draw the image within


Returns

void
