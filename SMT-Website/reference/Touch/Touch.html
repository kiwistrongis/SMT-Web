Name
Touch
Description

Touch has state information of one touch and extends TuioCursor.

Syntax

Touch(tuioCursor)
Touch(sessionID, cursorID, xCoord, yCoord)
Touch(ttime, sessionID, cursorID, xCoord, yCoord)

Parameters
tuioCursor:
- TuioCursor: The TUIO Cursor

sessionID:
- long: Session ID

cursorID:
- int: Cursor ID

xCoord:
- float: X Coordinate

yCoord:
- float: Y Coordinate

ttime:
- TuioTime: TuioTime
