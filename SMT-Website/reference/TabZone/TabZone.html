Name
TabZone
Description

TabZone is a container for other Zones, it has tabs along its top, allowing easy switching between different views. It optionally has close buttons on the tabs. Currently, each zone added to TabZone becomes its own Tab, and so to have multiple Zones within a tab, use another zone as a container and add that zone to TabZone.

Syntax

TabZone()
TabZone(name)
TabZone(x, y, width, height)
TabZone(name, x, y, width, height)
TabZone(name, x, y, width, height, tabHeight)
TabZone(closeButtons)
TabZone(name, closeButtons)
TabZone(x, y, width, height, closeButtons)
TabZone(name, x, y, width, height, closeButtons)
TabZone(name, x, y, width, height, tabHeight, closeButtons)

Parameters
name:
- String: The name of the zone

x:
- int: X-coordinate of the upper left corner of the zone

y:
- int: Y-coordinate of the upper left corner of the zone

width:
- int: Width of the zone

height:
- int: Height of the zone

tabHeight:
- int: The height of the tab

closeButtons:
- boolean: Display close/exit buttons for tabs
